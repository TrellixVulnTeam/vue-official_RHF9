### 过滤器

#### 		1.局部过滤器	

​				在当前组件内部使用过滤器

​				给某些数据 `添油加醋`

```javascript
// 声明
filters: {
    '过滤器的名字':function(val,a){
        // a就是lee，val就是当前的数据
    }
}
// 使用 管道符
数据 | 组件的名字('lee','jack')
```

#### 		2.全局过滤器

​				只要过滤器一创建，在任何组件中都能使用

```javascript
Vue.filter('过滤器的名字',function(val,a,b){})
在各个组件中都能使用
数据 | 过滤器的名字('lee','jack')
```
