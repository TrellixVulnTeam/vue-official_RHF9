### 过滤器

-   局部过滤器

    -   ```javascript
        只能在当前组件内部使用
        filters:function(val,a,b){
            // 添油加醋的处理
            return xxxx
        }
        ```

        

-   全局过滤器

    -   ```javascript
        // 声明+创建 在任何组件中都能使用
        Vue.filter('myTime',function(){
            // 添油加醋的处理
            return xxxx
        })
        ```
